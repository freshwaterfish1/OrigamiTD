Game Freezes

During gameplay the game might lag or freeze totaly. Game is unresponsive - gameplay is affected.

Hit the “r” key

This is coded as a hard reset to the current level. Although the game might not seem to respond to you, hitting the r key will start the reset as soon as the game can.
